# DBMS SQL Assignments

This repository contains SQL assignments based on a Student Management System.  
It covers fundamental Database Management System (DBMS) concepts using structured SQL queries.

---

## Overview

The project demonstrates core DBMS operations such as table creation, data insertion, querying, joins, subqueries, views, and aggregate functions.

---

## File Structure

- `01_create_tables.sql` → Create database and tables  
- `02_insert_data.sql` → Insert sample data  
- `03_select_queries.sql` → Basic SELECT queries  
- `04_joins.sql` → INNER, LEFT, RIGHT joins  
- `05_subqueries.sql` → Nested and subqueries  
- `06_views.sql` → Creating and using views  
- `07_aggregate_queries.sql` → COUNT, AVG, SUM, GROUP BY, HAVING  

---

## Technologies Used

- SQL

## Concepts Covered

- Data Definition Language (DDL)
- Data Manipulation Language (DML)
- Joins (Inner, Left, Right)
- Subqueries
- Views
- Aggregate Functions
- Grouping and Filtering (GROUP BY, HAVING)
